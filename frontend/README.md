# Фотостудия "Экспозиция" - Frontend

Фронтенд приложение для системы управления бронированием фотостудии. Проект выполнен на React с использованием TypeScript, React Router и Tailwind CSS.

## 🚀 Функционал

### Для пользователей:
- ✅ Регистрация и авторизация с JWT токенами
- 📸 Просмотр доступных студийных залов
- 📅 Бронирование залов на определённое время
- 🤖 ИИ-прогноз спроса на выбранную дату
- 💳 Оплата бронирований (карта, наличные, онлайн)
- 📋 Управление своими бронированиями
- 👤 Личный профиль

### Для администраторов:
- ➕ Добавление новых залов
- ✏️ Редактирование информации о залах
- 🗑️ Удаление залов
- 📊 Просмотр аналитики:
  - Общее количество пользователей
  - Общее количество залов
  - Общее количество бронирований
  - Общий доход студии

## 🛠️ Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **React Router 7** - маршрутизация
- **Tailwind CSS v4** - стилизация
- **Radix UI** - доступные UI компоненты
- **Lucide React** - иконки
- **date-fns** - работа с датами
- **Sonner** - уведомления

## 📦 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

Приложение будет доступно по адресу: `http://localhost:5173`

## 🔐 Демо-доступ

Для быстрого тестирования доступны демо-аккаунты:

- **Администратор**: логин `admin`, любой пароль

## 🔌 Подключение к бэкенду

На данный момент приложение работает с моковыми данными для демонстрации.

Для подключения к реальному бэкенду:
1. Откройте файл `/BACKEND_SETUP.md`
2. Следуйте инструкциям по настройке API

Основные шаги:
- Измените `BASE_URL` в `/src/app/services/api.ts`
- Раскомментируйте реальные API вызовы
- Настройте CORS на бэкенде

## 📝 Особенности реализации

- **Защищённые маршруты** - доступ только для авторизованных пользователей
- **Автоматическое обновление JWT токенов** - при истечении access токена
- **Проверка пересечений** - при создании бронирования проверяется свободное время
- **ИИ-прогнозы** - предсказание спроса на основе даты (выходные/будни)
- **Адаптивность** - приложение работает на всех устройствах

## 🔒 Безопасность

- JWT токены хранятся в localStorage
- Все запросы к защищённым эндпоинтам включают Authorization header
- Автоматический logout при невалидном токене
- Разделение прав доступа (пользователь/администратор)
