# Руководство пользователя

## Для клиента

### 1. Регистрация и вход

1. Откройте `/login`.
2. Перейдите на вкладку «Регистрация».
3. Заполните поля: имя, email, логин, пароль (минимум 8 символов).
4. Нажмите «Создать аккаунт».
5. После регистрации система автоматически авторизует вас.

### 2. Выбор студии

1. Откройте `/halls`.
2. Используйте поиск, фильтр цены и вместимости.
3. Выберите подходящий зал и перейдите в карточку.

### 3. Бронирование и оплата

1. Перейдите в карточку зала `/halls/:id`.
2. Выберите дату, кликните свободный слот или укажите время вручную.
3. При необходимости включите доп.услуги.
4. Нажмите «Создать бронирование» и перейдите к оплате.
5. В `/checkout` выберите способ оплаты и подтвердите действие.
6. Статус заказа изменится на `COMPLETED` после успешной оплаты.

### 3.1. История и отмена

1. Откройте `/my-bookings`.
2. Используйте поиск и фильтр по статусу.
3. Кнопка «Отменить» доступна только для `NEW`/`PENDING`/`CONFIRMED`.
4. Подтвердите отмену в диалоговом окне.

### 4. AI-прогноз загрузки

1. Перейдите на `/ai-insights`.
2. Выберите зал.
3. Укажите период (дата начала и дата окончания).
4. Нажмите «Построить heatmap».
5. Изучите карту загрузки и блок рекомендаций.

## Для менеджера/администратора

### 1. Расписание

1. Перейдите на `/manager/schedule`.
2. Укажите период для просмотра.
3. В timeline занятые слоты выделены отдельными карточками.
4. Нажмите на занятый слот, чтобы открыть боковую панель заказа.
5. Измените статус заказа (`PENDING`, `COMPLETED`, `CANCELLED`) при необходимости.

### 2. Админ-панель

1. Откройте `/admin-panel`.
2. Используйте вкладки:
   - Аналитика
   - Залы
   - Заказы
   - Пользователи
   - Логи аудита
   - Услуги
   - Акции
3. Для аудита событий используйте фильтры по дате, действию и поиску.

## Подсказки

- Если сессия истекла, повторно войдите через `/login`.
- Для корректной работы realtime и фоновых задач убедитесь, что backend сервисы запущены.
- При сетевых ошибках используйте кнопки «Обновить»/«Повторить».
